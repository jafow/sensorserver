# sensorserver
service for collecting and storing sensor data

## install
```bash
git clone <this repo>
```
### dependencies
(optional but recommended)
create a virtual env

```bash
python3 -m venv env
source env/bin/activate
```

```bash
pip3 install -r requirements.txt
```

## start the app
```bash
python3 startup.py
```

in a browser go to localhost:5000/temperature
